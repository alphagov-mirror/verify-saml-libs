The individual SAML libraries used by Verify have been combined in this repository to make managing dependencies easier.

Affected libraries:

* saml-extensions
* saml-security
* saml-utils
* saml-serializers
* saml-metadata-bindings
* saml-metadata-bindings-test
* trust-anchor

### Building the project

`./gradlew clean build`
